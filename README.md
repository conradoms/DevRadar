<h1> Projeto Devradar </h1>
 <p> O Devradar é um projeto que tem como finalidade a facilitar a busca por desenvolvedores que estão próximos a você. O sistema detecta sua localização geográfica utilizando o sistema de GPS do celular e então plota todos desenvolvedores próximos em um raio de 10KM no mapa nativo do smartphone, utilizando a API do GitHub para puxar informações do perfil cadastrado. </p>
  
 <p> No Mobile, a aplicação exibe o mapa aonde É possível ainda pesquisar os desenvolvedores por tecnologias. Este projeto possui também o recurso para trabalhar em tempo real. </p>

<p> Logo, quando um desenvolvedor é inserido na aplicação web, se uma das tecnologias cadastradas para ele estiver sendo pesquisada e ele estiver dentro do raio do usuário que está realizando a pesquisa, ele será apresentado junto aos resultados desta pesquisa. </p>
